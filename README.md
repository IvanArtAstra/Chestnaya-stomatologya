# Честная стоматология — сайт клиники

Иммерсивный сайт стоматологической клиники «Честная стоматология» (Пермь, ул. Танкистов, 68)
со scroll-погружением, бегущей лентой врачей, блогом и демо-админкой.

**Живая версия:** https://ivanartastra.github.io/Chestnaya-stomatologya/

## Возможности

- 🌊 Scroll-сцена «погружения» с параллаксом и шкалой глубины
- 🦷 Фирменные SVG-иконки (штрих 1.8), шрифты Unbounded / Manrope / Cairo
- 👨‍⚕️ Бегущая лента врачей → личные страницы с отзывами (`doctor.html?id=…`)
- ⭐ Реальные отзывы и рейтинг 5,0 из 2ГИС, девиз и акции из ВКонтакте
- 🗺 Яндекс-карта с меткой клиники + кнопка «Маршрут в 2ГИС»
- 📝 Блог с Markdown и живым превью
- 🛠 Демо-админка (`admin.html`): цены, карточки врачей, посты — localStorage
- 🌗 Тёмная и светлая темы, «жидкое стекло» (blur + saturate) в шапке и виджетах
- 🌍 Мультиязычность RU / EN / AR (с RTL)
- ♿ prefers-reduced-motion, работоспособность без JavaScript

## Стек

Чистые HTML/CSS/JS без сборки и зависимостей — быстрый старт (< 3 c по ТЗ),
хостится где угодно. Данные (цены, врачи, отзывы, посты) — в `site/db.js`;
на проде слой заменяется на REST API той же формы.

## Запуск локально

```bash
cd site && python3 -m http.server 8734
# → http://localhost:8734
```

или двойной клик по `Открыть сайт.command` (macOS).

## Структура

| Файл | Назначение |
|---|---|
| `site/index.html` | главная: услуги, калькулятор, команда, до/после, акции, лента, отзывы, контакты |
| `site/doctor.html` | личная страница врача (отзывы + публикации) |
| `site/admin.html` | админ-панель: цены, врачи, блог |
| `site/db.js` | мини-CMS: данные + localStorage |
| `site/i18n.js` | словарь RU/EN/AR |
| `site/md.js` | безопасный Markdown-рендерер |
